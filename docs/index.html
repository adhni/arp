ETC4500/5450 Advanced R Programming

R is widely used as a tool for data analysis and one of the most popular programming languages. This unit delves into R from the programming aspect instead of using it as a data analysis tool. You will learn a variety of programming paradigms and delve into the foundations of R as a programming language.

Learning outcomes

01

Build a solid understanding of the foundations of R as a programming language.

02

Work confidently across multiple paradigms, including functional and object-oriented programming.

03

Use R’s functional and object-oriented systems to design reusable, well-structured code.

04

Apply metaprogramming ideas to evaluate, generate, and extend code programmatically.
